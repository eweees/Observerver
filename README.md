# Time Server App

JavaFX-приложение с централизованным сервером времени, реализованным по паттерну **Observer**.  
Различные компоненты приложения реагируют на ход времени: отображают прошедшие секунды, запускают анимацию, выполняют визуальные эффекты и воспроизводят звуковые сигналы.

---

## Функции
- Централизованный сервер времени с шагом 1 секунда
- Запуск, остановка и сброс времени
- Отображение прошедшего времени в пользовательском интерфейсе
- Периодическая анимация объекта с настраиваемым интервалом
- Условное мигание панели через заданное количество секунд
- Воспроизведение звукового сигнала (будильника) с заданным интервалом
- Динамическое подключение и отключение компонентов-наблюдателей
- Потокобезопасное обновление интерфейса (JavaFX Application Thread)

---

## Архитектура
- Паттерн проектирования **Observer**
- `TimeServer` — источник времени (Subject)
- Компоненты (`TimeLabelComponent`, `AnimationComponent`, `ConditionalBlinkComponent`, `MusicPlayerComponent`) — наблюдатели (Observers)
- Слабая связанность компонентов и возможность лёгкого расширения системы

---

## Технологии
- Java 17
- JavaFX
- Maven
- `Timer` / `TimerTask`
